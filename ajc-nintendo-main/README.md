# ajc-nintendo

## Explication
Ce projet existe afin d'expérimenter et d'apprendre a utiliser git. Il s'agit en effet d'un rôle play entre une personne jouant le rôle de chef de projet et 3 autres de son équipe

## Cadre
Le chef de projet possède toutes les informations mais demande moins de travail pratique. Cependant il lui convient de gérer le répo git également

## Objectif 
L'objectif est de créer une suite de classe immitant une boutique vendant des consoles et des jeux
